# Melody DeBenedictis Artist Website

Monorepo for the Melody DeBenedictis fine art site.

## Workspace
- `client/` — React + TypeScript + Tailwind frontend
- `server/` — Express TypeScript backend stub for future API and image processing

## Getting started
1. `npm install`
2. `npm run dev`

## Notes
- The initial frontend is focused on site structure, navigation, gallery presentation, blog layout, artist bio, and commission/event exploration.
- Image management and backend storage will be added after the visual and functional pages are validated.
- Future work: full-resolution image uploads, responsive image generation, thumbnails, watermarking, and admin media management.
